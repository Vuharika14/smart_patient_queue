# 🏥 MediBook — Hospital Appointment Booking System

MediBook is a smart web-based platform that allows users to search hospitals, select doctors, book appointments, and receive instant token numbers — eliminating long waiting times in hospitals.

---

## 🚀 Features

* 🔍 Search hospitals by city, name, or specialty
* 🏥 View hospital details (facilities, doctors, reviews)
* 👨‍⚕️ Select doctor & department
* 📅 Full calendar-based appointment booking
* ⏰ Time slot selection with availability
* 🧾 Patient details form
* 💳 Multiple payment methods (Card, UPI, Net Banking UI)
* 🎫 Instant token generation after booking
* 🔐 Login & Registration system (UI)
* 🛠️ Admin dashboard (UI for managing hospitals & appointments)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend (Planned):** PHP / Flask

## 🎯 Problem Statement

Patients often face long waiting times in hospitals due to manual appointment systems. MediBook solves this by providing a digital booking system with time slots and token generation.
